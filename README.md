# Exploratory Data Analysis (EDA) Dashboard

## Description
This project focuses on Exploratory Data Analysis (EDA) using Power BI. The goal is to analyze a dataset and identify patterns, trends, distributions, and key insights through visualizations and statistical measures.

## Features
- Calculated Mean, Median, and Count
- Product Sales Analysis
- Payment Method Analysis
- Referral Source Analysis
- Order Status Distribution
- Sales Trend Visualization
- Outlier Observation
- Interactive Dashboard

## Tools Used
- Power BI
- Excel Dataset
- GitHub

## How to Run
1. Open Power BI Desktop
2. Load the dataset file (.xlsx)
3. Open the project dashboard (.pbix)
4. Explore the visualizations and insights
